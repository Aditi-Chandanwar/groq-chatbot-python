# groq-chatbot-python
A simple conversational AI chatbot built in Python using the Groq API and Llama 3.3. This project demonstrates environment variable management with .env, maintaining conversation history, and building a multi-turn interactive chat interface inspired by Anthropic's Claude API workflow

Groq Chatbot with Python

This project is a simple multi-turn chatbot built using the Groq API and Llama 3.3 70B Versatile model. It maintains conversation history and allows interactive chatting through Python.

Features
🤖 Conversational AI powered by Groq
💬 Multi-turn chat with conversation memory
🔐 Secure API key management using .env
🐍 Built with Python
🔄 Anthropic Claude API concepts adapted to Groq
📚 Beginner-friendly implementation
Tech Stack
Python
Groq API
Llama 3.3 70B Versatile
python-dotenv
Installation
pip install groq python-dotenv

Create a .env file:

GROQ_API_KEY=your_api_key_here

Run the chatbot:

python chatbot.py
Example
> What's 1+1?
2

> Explain quantum computing in one sentence.
Quantum computing leverages quantum mechanical principles to process information in ways that can outperform classical computers for certain tasks.
Learning Goals

This project was created while learning LLM application development and adapting concepts from Anthropic's Claude API course to the Groq API ecosystem.
